Started as a complimentary project for Database class assignment. Now used to learn and try out new things in VueJS and PHP. News stories fetched from [WorldNews API](https://worldnewsapi.com/).

## Technology used
- Frontend

- Backend
    - Plain PHP
    - MySQL
